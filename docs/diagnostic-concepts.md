## Implementation Boundary

The maintained registry and deterministic placeholder generators live in `packages/core/src/diagnostics/index.js`.

During intake, diagnostic concepts must use `implementation_state: "documentation-only"` or `implementation_state: "not-implemented"`. Executable diagnostics require a later PR with source provenance, contract fixtures, tests, and scientific caveat review.

Placeholder diagnostic outputs use the separate `diagnosticResult` contract with `status: "placeholder"` and `implementation_state: "placeholder"`. They may appear in validation report JSON only as caveated, deterministic research workflow artifacts.
