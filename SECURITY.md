# Security Policy

## Supported Scope

This repository is in early research-source scaffolding. Security reporting currently covers repository content, source code once imported, examples, schemas, and documentation.

## Reporting a Vulnerability

Please do not open public issues for vulnerabilities that could expose users, credentials, private data, restricted datasets, or deployment infrastructure.

Report privately to the repository steward through the GitHub security advisory workflow when available, or through the official contact channel designated by AI-Bio Synergy Holdings LLC.

## Data Safety

Do not submit:

- credentials, tokens, or secrets;
- personally identifying information;
- restricted dataset copies;
- regulated or export-controlled materials;
- private institutional data.

## Current Deployment Assumption

Until otherwise documented, COSMOS-CQA is not approved for production deployment or hosted public service operation under the research-only public license.
