Quickstart

Run the portal and research workbench locally.

The public portal and demo workbench are static files served by the repository workflow. The hosted demo path preloads the public sample Core Pack and keeps truth labels hidden unless dev mode is explicit. 

git clone https://github.com/AI-Bio-Synergy-Holdings-LLC/COSMOS-CQA.git
cd COSMOS-CQA
npm --prefix apps/web ci
npm --prefix apps/web run serve
