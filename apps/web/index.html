Quickstart

Run the portal and research workbench locally.

The public portal and demo workbench are static files served by the repository workflow. The workbench remains local-first and browser-based for review, export, and replay tasks. 

git clone https://github.com/AI-Bio-Synergy-Holdings-LLC/COSMOS-CQA.git
cd COSMOS-CQA
npm --prefix apps/web ci
npm --prefix apps/web run serve
