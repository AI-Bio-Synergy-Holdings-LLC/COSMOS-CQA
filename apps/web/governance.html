Governance

Stewarded public research infrastructure.

AI-Bio Synergy Holdings LLC owns and stewards COSMOS-CQA, including project direction, release approval, license policy, repository administration, and official research artifacts.

Decision model

Public feedback, issues, discussions, and pull requests are welcome, but project decisions are made by the steward. No public contribution or discussion creates an obligation to merge, release, support, or commercially license any feature.

Release policy

  • Version tag and release notes.
  • Source snapshot and schema versions.
  • Validation report and SBOM when dependencies are present.
  • License, notice, citation, and third-party data-use notes.
