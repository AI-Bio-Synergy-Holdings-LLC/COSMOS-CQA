# Original COSMOS Materials

This directory records the provenance of the original COSMOS materials used to seed the COSMOS-CQA rework.

Large raw exports are intentionally not committed in the initial repository scaffold. The original files are tracked by path, size, timestamp, and SHA-256 in `MANIFEST.md`.

Use this directory for:

- source inventory;
- extraction notes;
- canonicalization decisions;
- migration records;
- checksums.

Avoid committing large duplicate exports, raw zipped packages, or restricted third-party data unless the steward explicitly approves it.
